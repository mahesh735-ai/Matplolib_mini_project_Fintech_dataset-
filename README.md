# Matplolib_mini_project_Fintech_dataset-
This project analyzes FinTech sales data using Matplotlib to uncover trends, distributions, and relationships. It includes line, bar, scatter, histogram, box, pie, and heatmap visualizations, along with a dashboard, to generate meaningful business insights for decision-making.
# 📊 FinTech Sales & Customer Analysis (Matplotlib Project)

## 🎯 Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) using Matplotlib on a FinTech dataset. The goal is to analyze business performance, understand customer behavior, and extract meaningful insights through data visualization.

The project covers multiple visualization techniques such as line plots, bar charts, scatter plots, histograms, box plots, pie charts, heatmaps, and dashboard creation using subplots.

---

## 📂 Dataset Information
The dataset contains the following columns:
- **Date**: Daily transaction data
- **Category**: Business segment (Payments, Loans, Investments)
- **Region**: Geographical region
- **Sales**: Revenue generated
- **Profit**: Profit earned
- **Customers**: Number of users
- **Marketing Spend**: Advertising expenditure

---

## 📊 Key Analysis Performed

### 📈 Trend Analysis
- Sales trend over time using line plot
- Business growth using stackplot

### 📊 Category & Region Analysis
- Category-wise sales comparison (Bar chart)
- Region-wise contribution (Pie chart)

### 📉 Distribution Analysis
- Profit distribution using histogram
- Sales outlier detection using box plot

### 🔍 Relationship Analysis
- Sales vs Profit (Scatter plot)
- Marketing vs Sales (Scatter plot)

### 🧩 Dashboard Creation
- Combined multiple charts using subplots for executive view

### 🔥 Correlation Analysis
- Heatmap to identify relationships between variables

---

## 🧠 Key Insights
- Sales show an overall increasing trend indicating business growth.
- Strong correlation observed between marketing spend and sales.
- Certain categories contribute significantly higher revenue.
- Outliers detected in sales data indicate exceptional performance or anomalies.
- Profit is positively correlated with sales but not always proportional.
